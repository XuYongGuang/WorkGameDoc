# 03 Game Phases

# 总体目标

将单局游戏划分为五个阶段。

每个阶段提供新的挑战、机制与决策。

避免后期玩法重复。

---

# Phase 1 创业阶段

人数：1~10人

目标：

- 生存
- 完成首批项目
- 建立初始文化

解锁：

- 招聘系统
- 项目系统
- 基础事件

Boss：投资人审查

核心体验：

资源极度匮乏。

每一次招聘都是重大决策。

---

# Phase 2 小型公司阶段

人数：10~30人

目标：

- 建立部门
- 建立管理体系
- 初步形成Build

解锁：

- 部门系统
- 主管职位
- 文化科技树

Boss：核心员工离职潮

核心体验：

团队开始失控。

---

# Phase 3 成长公司阶段

人数：30~60人

目标：

- 建立组织结构
- 应对派系
- 扩张业务

解锁：

- 派系系统
- 中层管理
- 高级项目

Boss：办公室政治危机

核心体验：

人际关系成为主要问题。

---

# Phase 4 企业阶段

人数：60~100人

目标：

- 自动化管理
- 多部门协作
- 品牌建设

解锁：

- 自动化规则编辑器
- AI辅助系统
- 公关系统

Boss：舆论危机

核心体验：

管理开始依赖规则而非个人。

---

# Phase 5 自动化帝国阶段

人数：100+

目标：

- 构建稳定生态
- 实现高度自动化
- 完成终局挑战

解锁：

- AI管理层
- 全局规则系统
- 终局科技

Boss：AI接管事故

核心体验：

玩家管理规则。

规则管理公司。

---

# 阶段设计原则

每个阶段必须新增：

1. 新机制
2. 新风险
3. 新决策
4. 新Boss

禁止纯数值扩张。
